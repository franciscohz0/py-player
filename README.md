# PyPlayer

Un reproductor de música simple para escritorio hecho en Python

## Características

- Soporte para mp3, m4a, flac
- Lectura de metadatos
- Portadas embebidas

## Tecnologías usadas

- Python
- PySide6 (Qt Binding)
- Mutagen

## Dependencias

Instala las dependencias con:
```bash
pip install -r requeriments.txt
```

## Uso 

```bash
python music_player.py
```

## Nota

Proyecto realizado como práctica de Python y desarrollo de interfaces gráficas.
